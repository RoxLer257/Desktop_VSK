# Руководство использования

Перед использованием необходимо активировать **PractikaBD.sql** файл - там хранится База Данных приложения  
После выполнить подключение БД к приложению  
После выполнения этих действий можно пользоваться программой  

## Администратор

**Логин**: admin  
**Пароль**: admin  

**Доступные функции**: Добавление нового пользователя, удаление пользователя, добавление, удаление и редактирование полисов
